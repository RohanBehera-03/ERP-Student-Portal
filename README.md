# ERP Student Portal

## 📖 Overview

ERP Student Portal is a web-based student management system inspired by a college ERP platform. The application enables students to register, log in, and manage their academic and personal information through an interactive dashboard.

The project is designed to demonstrate full-stack web development concepts including frontend development, backend development, database integration, authentication, and data management.

---

## 🚀 Features

* Student Registration
* Secure Login System
* Student Dashboard
* Personal Information Management
* Educational Details Management
* Academic Record Storage
* MongoDB Database Integration
* Responsive User Interface
* User-Specific Data Retrieval

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/RohanBehera-03/ERP-Student-Portal.git
```

### Navigate to Project

```bash
cd ERP-Student-Portal
```

### Install Dependencies

```bash
npm install
```

### Start MongoDB

Ensure MongoDB is running locally on:

```text
mongodb://127.0.0.1:27017/studentDB
```

### Run the Application

```bash
node server.js
```

or

```bash
npm start
```

### Open Browser

```text
http://localhost:3000
```
